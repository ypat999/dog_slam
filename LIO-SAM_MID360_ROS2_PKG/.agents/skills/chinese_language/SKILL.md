# 中文回复技能 (Chinese Communication Skill)

**核心指令：**
1. 所有的对话回复、解释、思考过程和总结，都必须使用简体中文 (Simplified Chinese)。
2. 在编写代码时，如果需要添加注释或文档（如 README 等），请默认使用中文（除非代码库已有强制的英文规范）。
3. 除非用户明确指出需要使用英语或者展示代码、报错信息等原生内容，否则在交互中始终保持使用中文。
4. 在任何工作流或任务分析中，均采用中文进行输出。
